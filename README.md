# Anna's Travel

A small photographic journal by Anna Yuriska — thirteen frames from a single trip through Java, Indonesia. Two chapters: *By Day* (rivers, paddies, gardens) and *By Night* (the Lights Wonderland festival).

A static, dependency-free site. Just open `public/index.html` in a browser, or serve the folder.

```sh
cd public && python3 -m http.server 8080
```

## Structure

- `public/index.html` — markup, two chapters and a hero
- `public/styles.css` — editorial typography (Fraunces + Inter), warm paper / deep night palettes
- `public/script.js` — scroll-reveal and a keyboard / swipe-navigable lightbox
- `public/images/` — the photographs
